## Project Overview

This project presents an **interactive Netflix Analysis Dashboard** built using **Power BI**. The dashboard provides deep insights into Netflix content based on genre, release year, ratings, country, and content type (Movies & TV Shows).

The objective of this project is to demonstrate **data analysis, visualization, and business intelligence skills** using a real-world entertainment dataset.

---

## Dataset

* **Source:** Netflix Movies and TV Shows dataset
* **File Used:** `Netflix Analysis Dashboard.pbix`
* **Data Includes:**

  * Title
  * Type (Movie / TV Show)
  * Genre
  * Country
  * Release Year
  * Rating
  * Duration
